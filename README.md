# BMP Editor — Командний проєкт

## ▶️ Запуск
```bash
python main.py
```
Потрібен лише **Python 3.10+** — жодних pip-пакетів не потрібно.

---

## 📁 Структура проєкту

```
bmp_project/
├── main.py                   ← точка входу
├── core/
│   ├── bmp_generator.py      ← генерація BMP (3 методи × 3 палітри)
│   └── steganography.py      ← LSB стеганографія
├── data/
│   ├── storage.py            ← авторизація + JSON-база
│   └── users.json            ← створюється автоматично
└── ui/
    └── app.py                ← увесь інтерфейс (Tkinter)
```

---

## 👥 Розподіл обов'язків

| Учасник | Файл | Задача |
|---------|------|--------|
| **1** | `core/bmp_generator.py` | Генерація BMP, 3 методи, 3 палітри |
| **2** | `core/steganography.py` | Вкладання/витягування повідомлень LSB |
| **3** | `data/storage.py` + `ui/app.py` | Авторизація, сесії, весь UI |

---

## 🎨 Методи генерації

| Метод | Опис |
|-------|------|
| 🔮 Фрактал Мандельброта | Колір ← кількість ітерацій до виходу |
| ✨ Плазмовий ефект | Колір ← суперпозиція 4 синусоїд |
| 💧 Концентричні хвилі | Колір ← відстань від центру |

## 🎨 Палітри

| Палітра | Опис |
|---------|------|
| 🌈 Веселка | RGB через зсунуті синуси |
| 🔥 Вогонь | чорний → червоний → жовтий → білий |
| 🌊 Океан | темно-синій → блакитний → білий |

---

## 🔒 Алгоритм стеганографії (LSB)

```
Повідомлення: "Hi"  =  0x48 0x69
0x48  =  0  1  0  0  1  0  0  0
                ↓  ↓  ↓  ↓  ↓  ↓  ↓  ↓
Пікселі: [x&0xFE|0] [x&0xFE|1] [x&0xFE|0] ...
         (молодший біт кожного байту замінюється)

Маркери: 0xABCDEF01 ... 0x01FEDCBA
```

## 👤 Авторизація

- Пароль зберігається у вигляді SHA-256 хешу
- Пам'ятає 3 останніх: файли, режими, повідомлення
- Автовідновлення сесії при вході
- Вихід зберігає поточні налаштування
